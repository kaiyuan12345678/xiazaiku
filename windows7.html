Windows 7

Windows 7是由微软公司（Microsoft）2009年10月22日发布的桌面端操作系统，内核版本号为Windows NT 6.1。

Windows 7可供家庭及商业工作环境的笔记本电脑、多媒体中心等使用。Windows 7继承了包括Aero风格等多项功能，并且在此基础上增添了其他功能。

Windows 7发布于2009年10月，2015年结束主流支持，2020年结束外延支持，随后微软上线了ESU，允许通过付费的方式订阅安全补丁，2023年1月10日，Windows 7操作系统结束ESU（付费外延扩展支持）。
